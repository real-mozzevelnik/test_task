Компиляция:
    g++ main.cpp -o test

    или

    make app

Запуск:
    Windows:
        start test
    Linux:
        ./test

Файл Product.h:
    Содержит данную по условию структуру Product
Файл AVL_Tree.h:
    Содержит реализацию АВЛ-дерева, выбранного для хранения данных.
Файл ProductsImpl.h:
    Содержит класс ProductsImpl, методы которого необходимо было реализовать.
Файл main.cpp:
    Содержит тесты.
